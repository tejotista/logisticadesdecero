# Web de Consultoría Logística (Plantilla Estática)

Este repositorio contiene el código fuente de una página web estática (landing page) diseñada para ofrecer servicios de consultoría logística enfocados en emprendedores y organizaciones que están iniciando sus actividades.

El objetivo de este diseño es transmitir la idea de que la logística se puede implementar desde cero, paso a paso, sin necesidad de recurrir a procesos complejos o software costoso en las etapas iniciales.

## Tecnologías Utilizadas

El proyecto está construido con tecnologías web fundamentales, sin frameworks pesados, para garantizar que sea fácil de entender, modificar y alojar de forma gratuita:

*   **HTML5:** Estructura de la información (`index.html`).
*   **CSS3:** Estilos visuales, grillas y diseño responsivo (`style.css`).

## Estructura del Proyecto

El repositorio consta de dos archivos principales:

*   `index.html`: Contiene toda la redacción (copywriting), la estructura de navegación y las secciones (Inicio, Enfoque, Servicios y Contacto).
*   `style.css`: Define la paleta de colores, la tipografía y el comportamiento visual de la página en diferentes dispositivos.

## 🚀 Cómo utilizar este código

Si querés estudiar este código o usarlo como base para tu propia web, el proceso es muy directo. Al ser un sitio estático, no requiere instalación de servidores locales (como Node.js o bases de datos).

**Paso a paso:**
1.  **Descargar el código:** Podés clonar este repositorio usando Git (`git clone [URL-del-repositorio]`) o hacer clic en el botón verde "Code" y seleccionar "Download ZIP".
2.  **Descomprimir:** Si descargaste el ZIP, extraé los archivos en una carpeta de tu computadora.
3.  **Visualizar:** Hacé doble clic en el archivo `index.html`. Se abrirá en tu navegador web predeterminado para que veas el diseño funcionando localmente.

## Cómo adaptar el contenido

Para personalizar la web para tu propio proyecto:
1.  Abrí `index.html` en cualquier editor de texto o código (como Bloc de notas, VS Code o directamente en GitHub) y modificá los textos que están entre las etiquetas (por ejemplo, cambiá el contenido dentro de los `<h1>`, `<p>`, etc.).
2.  Para modificar la paleta de colores, abrí `style.css` y alterá los valores hexadecimales en la sección `:root` al principio del archivo.
